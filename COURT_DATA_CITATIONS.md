## Citation Format for User Display

When serving data to users, display citations as:

```
Source: [Organization Name], "[Document Title]" (Date)
Link: [URL]
```

Example:
```
Source: California Judicial Council, "2024 Court Statistics Report" (Dec 2024)
Link: courts.ca.gov/sites/default/files/courts/default/2024-12/2024-court-statistics-report.pdf
```

---

## Verification Protocol

Before serving any data point to users:

1. ✅ Check if source URL is still active
2. ✅ Verify data hasn't been superseded
3. ✅ Cross-reference with official court website
4. ✅ Note last verification date
5. ✅ Flag if data is >6 months old
