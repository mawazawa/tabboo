Before starting any complex task, I must "rehydrate" my context using the Knowledge Graph.

*   **Step 1**: Identify the active Linear Ticket or Feature Context.
*   **Step 2**: Query Neo4j/Memory MCP for `(:Issue {id: ...})-[:RELATES_TO]->(:Component)`.
*   **Step 3**: Read the *summaries* of those components, not the full code, to respect the token budget.
